# Elixir Learn

A collection of utilities to learn Elixir

## Setup

To run this program you to need setup an Erlang+Elixir environment, as it is described here:
https://elixir-lang.org/install.html

-- OR --

You may use the provided "Dev Container" environment: https://code.visualstudio.com/docs/devcontainers/tutorial


### Download dependencies

    mix deps.get

### Compile

    mix compile

### Build & Run

    mix run

## Run tests

    mix test

## Run format

    mix format

## Interactive Elixir

    iex -S mix

## Generate docs

    mix docs
